Merge video and audio files with ffmpeg in Linux Terminal.  
Commands:  
````  
nano input.txt  
# input.txt content:  
# file 'file_1.mkv'  
# file 'file_2.mkv'  
ffmpeg -f concat -i input.txt -c copy output.mkv
